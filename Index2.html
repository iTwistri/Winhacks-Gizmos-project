Gizmo Fun

WinHacks Project

By:Evan Helgerman and Julian Buiza

This is a home page where you can choose what gizmo you want to play! Gizmos are a fun interactive way to learn.


Boiling Point

This game is to show what the elements boiling points are, it is used to help you learn elements boiling points and find a connection between the selected elements boiling points


Projectile Motion

This game is to show the projectile motion of diffrent hit speeds and angles, it is used to learn more about projectile motion and how the angles and hit speeds make a diffrence
